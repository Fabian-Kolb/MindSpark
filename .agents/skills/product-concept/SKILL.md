# App Concept: MindSpark AI Productivity Organizer

## When to use this skill
- Use this when making product-level decisions, planning new features, or modifying core app structures.
- This is helpful for understanding the high-level vision and ensuring the agent aligns with the "Luminous Sanctuary" philosophy of replacing chaotic lists with calm, structured "Spaces".

## How to use it

### Core Features to Keep in Mind
1.  **Focus Dashboard:** A central hub showing the daily suggestions, quick trackers, and a comprehensive overview of the day's focus.
2.  **Spaces Overview:** A section to manage isolated project areas (Uni, Work, Personal) to minimize context switching.
3.  **Productivity Tracker (Segmented Bar):** Advanced tracking functionality capturing stats, time, and focus segments visually.
4.  **Quick Brain-Dump:** A globally accessible floating action that allows the user to quickly capture passing thoughts without losing the context of their current task.
